三,数组

--1,概念
    英文名字叫Array,标志是[]
    作用是用来存储多个  相同类型的  数据.
--2,特点
    数组中的每个元素,都有下标,默认从0开始.
    数组一旦创建，长度不可变.
--3,创建数组:  数组类型  数组名 = 数组对象;
    --一般分为动态初始化和静态初始化
    --动态初始化
        `int[] a = new int[5];`
    --静态初始化
        `int[] b = new int[]{9,8,7,6,5};`
        `int[] c = {9,8,7,6,5};`
